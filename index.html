How It Works ?
Admin will have to login with the credentials to access the data / details of the vehicles
CCTV camera capturing the vehicles and directly send video footage to server.
Deep Learning model output send to the backend then OCR (Optical Character Recognition) extracts the text from the Number plate image and then matches with the Database of Organization
If the vehicle found unregister with the database then Immediately Alert the admin via mail
On Dashboard, Admin can see the CCTV camera footage and updated Records of the vehicles entering in his premises
At Backend Live feed go through the Deep Learning model and processing start on video frame by frame.
if vehicle number plate is registerd with the database then it let the vehicle pass or just show the details on the admin dashboard.
